Guie o bloco verde com o mouse, não deixando a bola tocar a linha vermelha.
Quando da a tela de Game Over e você clicar no botão verde, você voltar para a tela inicial.

Este foi o meu quarto projeto do jogo Pong, aprendi a fazer a tela inicial com ensinetecnologia, essa é a ultima versão, criei 3 jogos de pong em 3 dias, agora adicionei tela de game over.
